## 1.เครื่องผลิตปีศาจ ผู้กล้าไปพบเครื่องผลิตปีศาจที่กำลังทำงานอยู่จึงต้องรีบหยุดเครื่องก่อนที่ปีศาจจะออกมาไปมากกว่านี้ ซึ่งที่เครื่องผลิตมีคำสั่งเขียนไว้ดังนี้ ``` do { Console.WriteLine("Summon a monster!!"); } while (true); ``` จงแก้ไขคำสั่งด้านบนให้เครื่องหยุดทำการผลิตปีศาจทันที โดยไม่ให้แก้ไขบรรทัด while(true); --- ## 2.เครื่องโชว์สัตว์เลี้ยง เจ้าของร้านขายสัตว์เลี้ยง ต้องการนำแมวกับหมามาโชว์ตัวให้ลูกค้าดู จึงได้มาจ้างให้คุณเขียนโปรแกรมให้ โดยโปรแกรมจะทำการแสดงแมวให้ลูกค้าดูก่อน แล้วถึงจะถามว่าอยากดูหมาบ้างไหม? 1. ถ้าลูกค้าขอดูหมา โปรแกรมจะโชว์น้องหมาให้ดู แล้วค่อยถามต่อว่าอยากดูแมวบ้างต่อไหม? สลับไปเรื่อยๆ 2. ถ้าลูกค้าไม่ขอดูต่อ โปรแกรมจะบอกบอกลาลูกค้าแล้วปิดตัวเองลง ตัวอย่าง ``` Here a cute cat Do you want to see a dog? yes Here an awesome dog Do you want to see a cat? yes Here a cute cat Do you want to see a dog? no Good bye :) ```
